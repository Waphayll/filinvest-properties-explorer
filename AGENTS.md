# Project: AFAIS - As Far As I See

## Design Principles
- **Aesthetic**: Follows the "Editorial Aesthetic" design recipe with a Deep Blue and Light Slate high-contrast color scheme.
- **Typography**: Uses `Cormorant Garamond` (italic/bold) for headings and `Inter` for functional UI.
- **Layout**: Features a structured editorial header, a ledger-style interactive sidebar, and a full-bleed map with architectural overlays.
- **Map Interaction**: Uses OpenStreetMap (Leaflet) with a custom grayscale filter and interactive property registry overlay.

## Data Structure
- All projects are categorized by `Brand` (Product Line): Filigree, Aspire, Prestige, and Office Spaces.
- Coordinates for Alabang, Clark, and San Mateo locations.
